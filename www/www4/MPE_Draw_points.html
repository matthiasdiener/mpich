MPE_Draw_points

 Draws points on an X Windows display

Synopsis

#include <stdio.h>
#include "mpe.h"
int MPE_Draw_points( handle, points, npoints )
MPE_XGraph handle;
MPE_Point *points;
int npoints;

Input Parameters

handle MPE graphics handle
points list of points to draw
npoints number of points to draw

Location:mpe_graphics.c
