MPE_Describe_state

 Create log record describing a state

Synopsis

#include <stdio.h>
#include "mpe.h"
int MPE_Describe_state( start, end, name, color )
int start, end;
char *name, *color;

Notes

Adds string containing a state def to the logfile. The format of thedef is (LOG_STATE_DEF) 0 sevent eevent 0 0 "color" "name".
