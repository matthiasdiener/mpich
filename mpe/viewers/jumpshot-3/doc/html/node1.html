Background

Assume one is trying to view a slog file called sppm_ic2a.slog1, which is generated from a run of the MPI program, sppm, on IBM's SP with shared memory nodes. The logfile is located in the directory ~/jumpshot-3/logfiles/. Also assume that Jumpshot-3 has been successfully installed2. Typing the full pathname to the jumpshot script which is located at ~/jumsphot-3/bin/ will bring up the Jumpshot-3.
